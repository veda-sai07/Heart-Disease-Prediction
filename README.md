# Smart AI-Based Cardiovascular Disease Prediction, Risk Assessment and Health Recommendation System

## Overview

This project predicts cardiovascular disease using multiple Machine Learning algorithms and automatically selects the best-performing model.

## Unique Features

* Automated Data Preprocessing
* Advanced Exploratory Data Analysis (EDA)
* Correlation Matrix Visualization
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)
* Automatic Best Model Selection
* Model Performance Leaderboard
* ROC-AUC Analysis
* Feature Importance Ranking
* Heart Health Score (0–100)
* Risk Meter (Low / Medium / High)
* BMI Calculator
* Personalized Health Recommendations
* Auto-Saved Graphs
* Execution Time Analysis
* Downloadable Patient Report
